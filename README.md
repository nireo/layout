# Keyboard Layout

## Default Layer (_DEF)

```
Q   | W   | E   | R   | T   |   | Y   | U   | I   | O   | P
A   | S   | D   | F   | G   |   | H   | J   | K   | L   | ;
Z   | X   | C   | V   | B   |   | N   | M   | ,   | .   | /
            ESC | SPC | NAV |   | SYM | LSFT| RIV
```

## Symbol Layer (_SYM)

```
!   | @   | #   | $   | %   |   | ^   | &   | *   | (   | )
-   | +   | =   | _   | [   |   | ]   | {   | }   | |   | 
~   | `   | <   | >   | ?   |   | /   | '   | "   | ;   | :
            ___ | ___ | ___ |   | ___ | ___ | ___
```

## Navigation Layer (_NAV)

```
STAB| SWIN| TABL| TABR| VIMD|   | VIMU| HOME| END | BSPC| DEL
CMD | ALT | CTRL| SHFT| ENT |   | ENT | LEFT| DOWN| UP  | RGHT
SPCL| SPCR| UNDO| REDO| TAB |   | TAB | PGUP| PGDN| '   | NUM
            ___ | ___ | ___ |   | ___ | ___ | ___
```
